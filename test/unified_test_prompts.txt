# Claude CLI MCP Server 統合テストプロンプト
# WSL/Windows両環境対応
# 
# 使用方法：
# このファイルの内容をそのままMCPツールで実行してください

## テスト開始

### 1. Claude CLI動作確認
test_claude_cliツールを実行してClaude CLIが正しく動作するか確認してください

### 2. 基本的な会話
execute_claudeツールで「こんにちは、私は山田太郎です」と送信してください

### 3. セッション継続性の確認
execute_claudeツールで「私の名前を覚えていますか？」と送信してください

### 4. 日本語処理の確認
execute_claudeツールで「日本の四季について簡単に説明してください」と送信してください

### 5. 実行履歴の確認
get_execution_historyツールで直近3件の履歴を取得してください

### 6. ファイルコンテキスト機能
execute_claude_with_contextツールを使って、README.mdファイルについて「このプロジェクトの主な特徴を3つ教えてください」と質問してください

### 7. 現在のセッション確認
get_current_sessionツールで現在のセッションIDを確認してください

### 8. コード関連の質問
execute_claudeツールで「Pythonでフィボナッチ数列の最初の10項を計算するコードを書いてください」と送信してください

### 9. セッションリセット
reset_sessionツールでセッションをリセットしてください

### 10. リセット後の確認
execute_claudeツールで「私の名前を覚えていますか？」と送信してください（リセットされているはずです）

### 11. エラーハンドリングテスト
execute_claude_with_contextツールで存在しないファイル「/tmp/not_exist_test_file.txt」について質問してみてください

### 12. 履歴クリア
clear_execution_historyツールで履歴をクリアしてください

## テスト完了

すべてのテストが正常に完了したか確認してください。
各ツールが適切な`tool_name`フィールドを含む応答を返すことも確認してください。
