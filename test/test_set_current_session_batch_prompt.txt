# set_current_sessionツール一括テストプロンプト

以下の11個のステップを順番に実行して、set_current_sessionツールの動作をテストしてください。各ステップの結果を記録しながら進めてください：

1. execute_claudeツールで「私は太郎です。富士山について教えてください」を送信
2. execute_claudeツールで「私の名前を覚えていますか？また、先ほど話した山の高さは？」を送信
3. get_current_sessionツールでセッションIDを取得（これを「セッション1」として記録）
4. reset_sessionツールでセッションをリセット
5. execute_claudeツールで「私は花子です。桜について教えてください」を送信
6. execute_claudeツールで「私の名前を覚えていますか？また、先ほど話した花の特徴は？」を送信
7. get_current_sessionツールでセッションIDを取得（これを「セッション2」として記録）
8. set_current_sessionツールで「セッション1」のIDを設定
9. execute_claudeツールで「私の名前と、最初に話した山についてもう一度教えてください」を送信（太郎・富士山の確認）
10. set_current_sessionツールで「セッション2」のIDを設定
11. execute_claudeツールで「私の名前と、最初に話した花についてもう一度教えてください」を送信（花子・桜の確認）

最後に、セッション切り替えが正しく動作したかどうかをまとめて報告してください。
